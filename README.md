## Algoritmos implementados
- Bubble Sort (`sort_bubble.py`)
- Selection Sort (`sort_selection.py`)
- Insertion Sort (`sort_insertion.py`)
- Quick Sort (`sort_template.py`)

## Decisiones de implementación
- Seguimos la estructura del archivo `sort_template.py`.
- Cada algoritmo avanza de a un paso en la función `step()`.
- En `init()` se reinicia todo el estado del algoritmo.
- Los swaps se realizan antes de devolver `swap=True` para sincronizar con la animación.
- Se controlaron los índices para evitar errores.
